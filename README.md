Sharks N Kitties is a multiplayer PVP game where players are divided into sharks and cats. Their aim is to defraud each other of their gold, and the stop each other by burning each other alive, with cardboard boxes acting as fire retardant shields. The game ends when one team is completely defrauded of their gold.

Controls:
Joystick - Move
Thrust phone folder in a flat plane - fire spew
Rotate phone in hand - cardboard box shield
